## Overview
Flickr API

## Init app
1/ Git clone the repository
```
https://github.com/audreylamy/flickr_api.git
```

2/ Install packages
```
npm install
```

3/ Launch app
```
npm start
```

4/ Go to http://localhost:3000/

## Technology stack

+ Front-end: ReactJS, React Router 
+ Other: Styled-components
+ Api: Flickr
